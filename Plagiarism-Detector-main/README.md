# Plagiarism Detector

A powerful multi-language plagiarism detection tool designed for compiler design analysis.

## Developed by **Priyam Gupta**

### Features

- **Multi-Language Support**: Detects plagiarism across C, C++, Java, and Python.
- **Advanced Analysis**: Uses Lexical Analysis, AST Comparison, and Control Flow Analysis.
- **Visual Reports**: Detailed similarity matrix, side-by-side code comparison, and matched block highlighting.
- **Modern UI**: Built with React, Tailwind CSS, and Shadcn UI for a premium experience.

### Tech Stack

- **Frontend**: React, TypeScript, Vite
- **Styling**: Tailwind CSS, Shadcn UI
- **Logic**: Custom comparison engine (Levenshtein distance, AST parsing simulation)

### Getting Started

1.  **Clone the repository**
    ```bash
    git clone <your-repo-url>
    cd logic-weaver
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Run the development server**
    ```bash
    npm run dev
    ```

4.  **Build for production**
    ```bash
    npm run build
    ```
