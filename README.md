## 🚀 Overview

**Plagrisam Detector** is a source code plagiarism detection project inspired by compiler design concepts.  
It focuses on analyzing submitted code, checking structural similarity, and presenting results through a modern and user-friendly interface.

This project is built to help understand how plagiarism detection can go beyond simple text matching by using compiler-based ideas such as token analysis, Abstract Syntax Trees, Control Flow Graphs, and similarity scoring.

---

## ✨ Key Features

- 📁 Code/file submission interface
- 🧠 Compiler-inspired plagiarism detection workflow
- 🔍 Code similarity analysis
- 🌳 AST-based structural comparison concept
- 🔁 CFG-based logical flow comparison concept
- 📊 Visual result dashboard
- 📈 Similarity score representation

## 🧩 Core Concept

The project idea is based on a compiler-style plagiarism detection pipeline:

```text
Source Code
   ↓
Lexical Analysis
   ↓
Token Normalization
   ↓
Syntax / AST Analysis
   ↓
CFG / Logical Flow Analysis
   ↓
Similarity Calculation
   ↓
Result Dashboard
